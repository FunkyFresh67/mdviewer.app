Powerful Features

Everything you need for professional documentation workflows

🧭 Smart Navigation System

Intelligent directory navigation with breadcrumbs, local file menus, and automatic linking to related documents. Navigate complex documentation structures effortlessly with parent/child relationships and subdirectory exploration.

✏️ Built-in Text Editor

Integrated markdown editor with full menu support including new, open, save, find/replace, and word wrap. Edit and preview in one unified interface with seamless workflow integration.

📄 Professional Export

Export to high-quality PDF and Microsoft Word formats with proper formatting preservation. Perfect for sharing documentation with teams or creating professional reports and presentations.

🎨 GitHub-Style Rendering

Professional rendering with responsive design, image support, syntax highlighting, and clean typography. Your markdown looks exactly as intended with consistent styling across all content.

🏢 Documentation Hub

Enterprise documentation management with team-centric organization, priority-based access, search capabilities, and dashboard generation for coordinating large documentation projects.

⚡ High Performance

Built on .NET 8 and WebView2 for fast, reliable performance. Handles large documents and complex directory structures with ease, plus developer tools access for debugging.

🔧 Developer Integration

External editor integration, recent files tracking, settings persistence, command-line support, and drag-and-drop functionality. Seamlessly integrates with your existing workflow.

🔌 MarkdownViewer.Controls — Embed MDViewer in Your App

New in v1.7.0: the full MDViewer engine is available as a drop-in WinForms UserControl. Add a complete markdown documentation browser — rendering, smart navigation, breadcrumbs, back-button history, and PDF/Word export — to any .NET 8 or .NET Framework 4.8 Windows application. Learn more →

🖼️ Rich Media Support

Proper image loading with relative path resolution, Git repository integration for asset linking, and responsive scaling for all media types within your documentation.

📋 System Requirements

  • Operating System: Windows 10 version 1809 or later
  • .NET Runtime: .NET 8 (included in self-contained versions)
  • WebView2: Automatically installed by Windows Update
  • Memory: 512 MB minimum, 1 GB recommended
  • Storage: 100 MB available space
  • Additional: Internet connection for initial WebView2 setup
